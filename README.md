## Travel expenses API

No swagger yet

The API will provide the ability to create, modify, and delete travel expense accounting. It will also be quite possible to use the API as a financial accounting in everyday life, setting up accounting periods as you like

This is a pet project and a kind of test API for the open public

---
API будет предоставлять  возможности создания, изменения, удаления учета расходов в путешествиях. Также вполне возможно будет использовать API как учет финансов  в обычной жизни, настраивая периоды учета как угодно

Это пет-проект и своего рода тестовое API на открытую публику
